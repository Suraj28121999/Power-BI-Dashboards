# Awesome Chocolate Sales Project

This repository contains a Power BI dashboard for Awesome Chocolates, providing insights into key metrics like sales, shipments, and profitability.

## Dashboard Features:
- **Total Sales Trends**: Monthly sales performance with slicers for switching between trends like total profit and total costs.
- **LOW BOX SHIPMENT (LBS)**: Track low-volume shipments using a dedicated LBS category with a gauge chart and KPI.
- **Profit Target Indicators**: Icons in the sales table reflect performance against profit targets.

![Dashboard Screenshot](./Awesome%20Chocolate%20Dashboard%20Pic.png)

## Files
- `Awesome_Chocolate_Sales_project.pbix`: The Power BI dashboard file.
- `Awesome Chocolate Dashboard Pic.png`: Screenshot of the dashboard.

## Getting Started
To view the dashboard:
1. Clone this repository.
2. Open `Awesome_Chocolate_Sales_project.pbix` in Power BI Desktop.
